# Редактор отчёта по СИМ

## Исправление белого экрана на GitHub Pages

1. Загрузите содержимое этой папки в корень репозитория с заменой старых файлов.
2. Откройте **Settings → Pages**.
3. В **Source** выберите **GitHub Actions**.
4. Сделайте push в ветку `main` и дождитесь зелёной галочки во вкладке **Actions**.
5. Откройте сайт и нажмите `Ctrl/Cmd + Shift + R`.

В `vite.config.js` используется `base: './'`, поэтому JS, CSS и изображения открываются корректно в подпапке GitHub Pages.

## Moscow Sans

Положите лицензированный файл шрифта в `src/fonts/` и переименуйте его в один из вариантов:

- `MoscowSans.woff2` — предпочтительно;
- `MoscowSans.ttf`.

Подключение уже прописано в начале `src/styles.css`. После добавления файла сделайте новый commit и push.
