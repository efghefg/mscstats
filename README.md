# Редактор отчёта по СИМ

Веб-редактор трёх слайдов из исходного PDF. Макет использует оригинальные страницы как 1920×1080 фон и поверх них размещает редактируемые значения. Такой способ обеспечивает максимальное визуальное совпадение экспортируемых PNG с исходником.

## Запуск

```bash
npm install
cp .env.example .env
npm run dev
```

Без `.env` приложение работает в локальном режиме через `localStorage`.

## Supabase

1. Создайте проект Supabase.
2. Выполните `supabase.sql` в SQL Editor.
3. Заполните `.env`.
4. Перезапустите `npm run dev`.

Каждый слайд хранится отдельной строкой по ключу `(report_id, slide_no)`. Кнопка «Сохранить слайд» делает upsert, «Обновить данные» заново загружает конкретный слайд.

## Moscow Sans

Файлы шрифта намеренно не включены. Положите лицензированные файлы в `public/assets/fonts/` и раскомментируйте пути в `src/styles.css`:

- `MoscowSans-Regular.woff2`
- `MoscowSans-Medium.woff2`
- `MoscowSans-Bold.woff2`

## Экспорт

PNG формируется в размере 1920×1080 через `html-to-image`. Можно скачать текущий слайд или ZIP со всеми тремя.
